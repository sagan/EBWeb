#	◎このファイルの見方
#		GAIJI	EPWINGの外字(書籍定義文字)です。hXXXXは半角、zXXXXは全角を表します。
#		unicode	対応するUnicodeです。uXXXX(16進数)で表記します。
#			2文字以上の合字の場合、uXXXX,uXXXXのようにカンマ区切りで続けて記述します。
#		ascii	Unicodeが定義されない場合の代替文字です。EBWinではクリップボードコピーで使用します。
#		#で始まる行はコメントです。行内の # から末尾まではコメントです。
#		[未使用]	『広辞苑』CD-ROM中で出現しない未使用の外字です。
#		[漢字辞書]	漢字辞書のみで使用され、『広辞苑』本文では出現しない外字です。
#
#	◎外字調査について
#		使用したCD-ROMは、富士通スーパー統合辞書2001の『広辞苑第五版』です。
#		『広辞苑』本文に現れる全角外字1675個を重点的に調査しています。
#		漢字辞書の外字は、出現頻度が7以上の漢字を調査しています。
#
#		『広辞苑第5版』全角外字総数	5642個	A121-DD22
#		------------------------------------------------------------
#		半角外字と重複(A121-A331)	 205個
#		未使用の外字		 	 362個
#		「漢字辞書」のみ出現		3400個	←出現頻度7以上を調査
#		『広辞苑』本文の外字		1675個	←全漢字を調査
#		------------------------------------------------------------
#
#		Unicodeが見つけられなかった漢字は × を付記しています。
#		CJK漢字拡張-Aの文字は (Ext-A)、CJK漢字拡張-Bの文字は (Ext-B)とコメントしています。
#		(Ext-A)*、(Ext-B)* のように*のついた文字は、JIS2004に収録されているため、
#		MSゴシック・明朝ver5.0やメイリオフォント環境なら表示することができます。
#		CJK漢字拡張-A,Bの全ての文字を表示するには、MingLiU-ExtBなどのフォントが必要です。
#
#	◎二次使用について
#		このファイルの一部、または全部を、自由に改変、再利用、再配布することを許可します。
#
#	◎使用上の注意
#		WindowsXP SP2のMS UI Gothicでの使用を想定しています。
